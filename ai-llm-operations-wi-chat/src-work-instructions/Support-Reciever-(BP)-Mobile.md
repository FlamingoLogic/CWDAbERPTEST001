# 🏷️ Support Receiver (BP) – Mobile View

*Matt Law | Last Updated: 2025-05-07*

This document defines the **Support Receiver (BP) – Mobile View** functionality, navigation, tips, and related tools within the organisation’s systems and processes.

---

### 📚 Description
The **Support Receiver – Mobile View** tool provides a simplified version of the standard Support Receiver window, designed specifically for mobile and tablet use. It surfaces only the most critical client information in a single screen layout.

This view references the same underlying data as the full desktop window, ensuring consistency across devices.

---

### 🧭 Navigation
- Access via: Support Receiver (BP) window
- Or: Open from saved searches or linked dashboards when using a mobile device

---

### 🛠️ Functionality
- Displays core participant data including name, alerts, gender, disability info, and decision-making capacity
- Consent alerts and risk alerts appear prominently
- Embedded image support (participant photo)
- Editable fields for mobile users where permitted

---

### 🎯 Tips
- Best used on mobile or tablet devices when full navigation isn't practical
- Any edits made here update the same record as in the full Support Receiver window
- Use for quick reference or light updates while on-site
